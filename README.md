# Homebrew Tap

Provides tools from `github.com/k14s` via [Homebrew](http://brew.sh/) package.

To install a package:

```bash
$ brew tap k14s/tap
$ brew install ytt
$ brew install kbld
$ brew install kapp
$ brew install kwt
```

For Linux see [https://k14s.io](https://k14s.io) for instructions.
